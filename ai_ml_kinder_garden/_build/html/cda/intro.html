Introduction to Machine Learning#

Linear Regression#

Linear Regression Model with Least Mean Square Method

Assume \(y\) is a linear function of \(x\) features plus some noise \(\epsilon\)

\[ y = \theta_0 + \theta_1x_1 + \theta_2x_2 + ... + \theta_nx_n + \epsilon\]
\[ y = \theta^Tx + \epsilon\]

Note

  • m - number of data points.

  • n - number of features.

  • \(\theta_0\) is intercept.

  • \(\theta_i\) is coefficient for ith feature.

  • \(\epsilon\) is an error model as \(\mathbb{N(0, \sigma^2)}\)

Least Square Method#

Given m data points, find \(\theta\) which minimizes the mean square error.

\[ \hat{\theta} = arg\min_{\theta} L(\theta) = \frac{1}{m}\sum_i^m (y^i - \theta^Tx^i)^2\]

We find the optimal theta by setting gradient = 0.

\[ \partial{L\theta} = -\frac{2}{m} \sum_i^m (y^i - \theta^Tx^i)x^i = 0 \]
\[ \partial{L\theta} = -\frac{2}{m} \sum_i^m (y^ix^i - \theta^Tx^ix^i) = 0 \]

We can use the matrix notation for the same as follows :

\[ \partial{L\theta} = -\frac{2}{m} (Xy - XX^T\theta) = 0 \]
\[ XX^T\hat{\theta} = Xy\]
\[ \hat{\theta} = (XX^T)^{-1}Xy\]

Warning

As m, n grow larger \(\hat{\theta} = (XX^T)^{-1}Xy\) becomes very expensive to compute. Hence we use a different algorithm called Gradient Descent to solve this optimization problem.

Probabilistic view of LMS#

Why does Least Mean Square error work ?

  • Assuming the error \(\epsilon\) follows a Gaussian distribution \(\mathbb{N(0, \sigma)}\)

\[ p(y^i | x^i; \theta) = \frac{1}{\sqrt{2\pi}\sigma}\exp(-\frac{(y^i - \theta^Tx^i)^2}{2\sigma^2})\]

  • Assuming independence between probabilities of each event, we can calculate LOG Likelihood

  • Likelihood

\[ L(\theta) = \prod_i^m p(y^i | x^i; \theta) = \bigg(\frac{1}{\sqrt{2\pi}\sigma}\bigg)^m\exp(-\frac{(y^i - \theta^Tx^i)^2}{2\sigma^2})\]

  • Log Likelihood

\[ log(L(\theta)) = \prod_i^m p(y^i | x^i; \theta) = m\bigg(\frac{1}{\sqrt{2\pi}\sigma}\bigg) - \frac{1}{2\sigma^2}\sum_i^m{(y^i - \theta^Tx^i)^2}\]

Note

\(log(L(\theta))\) is maximized when \(\frac{1}{2\sigma^2}\sum_i^m{(y^i - \theta^Tx^i)^2}\) is minimized.

\(\therefore\) LMS works for Linear Regression.

Gradient Descent#

Gradient Descent algorithm starts by selecting an arbitary \(\theta\) and slowly updating the parameter by using the gradient of the parameter.

We repeat this process until we exceed iterations or convergence.

\[ \hat\theta^{t+1} = \hat{\theta^t} + \frac{\alpha}{m}\sum_i^m (y^i - {{\theta}^t}^Tx^i)x^i\]

Note

  • \(x^i\) is a vector in \(\mathbb{R^n}\)

  • \(\alpha\) - Learning Rate of gradient descent algorithm.

  • \(t\) - Iteration number

  • \(\theta^t\) - Learned parameter after iteration t.

Case Study#

You want to buy a new house, and want to price the house based on similar houses.

House dataset uses the following features

  1. Area

  2. #bedrooms

  3. #bathrooms

  4. #stories

  5. Sales Price (Prediction)

  6. etc.

We can use Linear Regression to solve this problem.

class GradientDescent:
    _FEATURES = [
            "LotArea",
            "1stFlrSF",
            "2ndFlrSF", 
            "FullBath",
            "YearBuilt",
            "BsmtFinSF1",
            "WoodDeckSF",
            "OpenPorchSF",
            "PoolArea",
            "MiscVal"
    ]
    _REQUIRED_FIELDS = [
            "SalePrice"
        ] + _FEATURES

    COEFFICIENTS = [
        "Intercept"
    ] + _FEATURES

    def _init_X_Y(self, housing_np):
        X1 = housing_np[:, 1:]
        return np.hstack([np.ones((X1.shape[0], 1)), X1]), housing_np[:, 0]

    def __init__(self):        
        self.housing_pd_train = pd.read_csv("../datasets/housing/train.csv", delimiter=",")[GradientDescent._REQUIRED_FIELDS]
        self.X_train, self.Y_train = self._init_X_Y(self.housing_pd_train.to_numpy())
        pass
        
    def _predict(self, theta, X):
        return theta@X.T
    
    def _gradient(self, theta, X, Y):
       return -2* (Y - self._predict(theta, X))@X / Y.shape[0]  
    
    def _loss(self, theta, X, Y):
        return np.linalg.norm(Y - self._predict(theta, X))**2 / X.shape[0]
        
    def gradient_descent(self, lr = 1e-9, tol = 1e-2, max_itr = 2000):
        X, Y = self.X_train, self.Y_train
        train_loss = []
        theta = np.ones(X.shape[1])
        g = self._gradient(theta, X, Y)
        train_loss.append(self._loss(theta, X, Y))
        while (np.linalg.norm(g) > tol and len(train_loss) < max_itr):
            theta = theta - lr * g
            g = self._gradient(theta, X, Y)
            train_loss.append(self._loss(theta, X, Y))
        
        self.plot_summary(theta, train_loss, X, Y)    

        df = pd.DataFrame(theta)
        df.index = GradientDescent.COEFFICIENTS
        df.columns = ["Coefficient"]
        return df

    def plot_summary(self, theta, train_loss, X, Y):
        fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 6))

        ax1.plot(train_loss, 'b-', label = "train loss")
        ax1.set_title('Training Loss decreasing with each iteration')
        ax1.set_xlabel('x')
        ax1.set_ylabel('Loss')
        ax1.legend()
        ax1.grid(True)
        
        ax2.plot(Y, label = 'Original')
        ax2.plot(self._predict(theta, X), label = 'Predicted')
        ax2.set_title('Original vs predicted values')
        ax2.set_ylabel('Housing Price')
        ax2.legend()
        ax2.grid(True)
        
        plt.tight_layout()
        
        plt.show()

gd = GradientDescent()
